• git clone path-to-repository
  • git status - статус текущего локального репозитория
  • git add path-to-file - добавить файл под контроль ГИТА
  • git commit -am "added new file index.html" - зафиксировать изменения и дать им комментарий
  • git push - отправить все закоммиченные изменения в репозиторий на сервер
  • git pull - получить все закоммиченные изменения с репозитория на сервер
